About Module 2

The TOEFL iBT Reading section uses an adaptive format. Module 2 has two possible paths:

  • Hard Path — focuses on academic reading content with higher-level comprehension tasks.
  • Easy Path — focuses on everyday reading content with practical comprehension tasks.

Both paths include the same task types: Complete the Words and reading comprehension questions. A new timer will begin when Module 2 starts.
